# Stable-Tail GNN 10seed paper results package

This folder collects 10seed formal experiment outputs and table sources.

- Model seeds: 0-9.
- PyVRP solver seeds: 0-9.
- Main downstream split: Split B.
- Main edge-risk matrix: `w_floor = 0.01 + 0.99w`, `R_ij = w_floor * max(S_i_norm, S_j_norm)`.
- Original 5seed outputs are left untouched.
